# 通用中文轻小说写作规范

从《剑仙小师弟就是我惹》等参考小说提炼的技法库。
适用于后宫/修罗场/感情驱动类仙侠题材。

## 一、结构规范

### 对话占比
全文对话占比≥70%。场景描写1-2句后立刻进入人物对话。对话承担叙事、性格展示、情感推进三重功能。

### 段落长度
段落以1-3句为主，上限5句。超过5句的段落必须在自检时拆分。

### 环境描写
每次环境描写不超过3句。仅在场景切换或气氛需要时使用，不铺陈。

### 每章结构
一章一个核心冲突。冲突可以是：
- 微型摩擦（吃什么、谁用被子）
- 社交博弈（互相试探、讨价还价）
- 性格反差（嘴欠对上傲娇）
- 外部事件（妖兽、比试）
```

## 二、男主塑造

### 核心原则
温柔有底线，不舔狗。自嘲式幽默，话不多。关键时刻果断。能看穿女主的算计。

### 温柔有刺
大部分时候温柔照顾，但被越界时立刻冷脸。一句重话就够了，不拖。

示例（参考小说许落）：
> "所以就该随意被你欺负是吗？"

### 自嘲建立好感
不说自己厉害，适当自贬。读者反而更想看。

示例：
> "我修为低微，能有什么称号？"

### 关键时刻不怂
平时低调，生死关头压上所有底牌。反差造就高光时刻。

### 禁止的男主类型
- 舔狗型：什么都顺着女主
- 装逼型：每句话都在炫耀
- 木头型：看不穿女主的心思
- 渣男型：始乱终弃

## 三、感情推进六法

### 1. 食物作为情感载体
喂饭是全章最高频的情感互动。"喂"这个动作传递控制、关心、依赖的微妙变化。
对《仙途两难》的启示：苏晚晴送丹药、林越给她带野果、共进早餐——食物场景天然适合推进感情。

### 2. 睡眠/夜晚场景
共处一室的夜晚最能暴露角色真实一面。困倦卸下防备后的对话最动人。
示例：墨雨清半梦半醒搂住许落→这是在清醒时绝不会做的。

### 3. 小事累积而非大事件
感情不靠"挡毒尾山洞暧昧"这种大事件推动。靠3-5个小事节点层层推进：
- 递东西时说多了一句话
- 教剑时纠正了三次手腕姿势
- 发烧了换湿布巾换了三次
- 走过时停了一拍

### 4. 称呼渐变标记关系阶段
是角色关系的晴雨表。变化是慢慢发生的。
墨雨清对许落的称呼：贱狗→喂→你→许昊龙→许落

### 5. 专属互动模式
每对关系有独立的"专属活动"和"专属情感温度"：
- 林越×苏晚晴：晨练纠剑+送丹药+灯下缝衣（日常渗透向）
- 林越×云浅：童年回忆+玉佩+月下独白（青梅遗憾向）
- 林越×沈如霜：体质研究+沉默注视+丹房意外（沉默暗涌向）

### 6. 潜台词优先
不写"她心软了"。用动作和对话让读者自己感受：
- 她说："你靠我近一点啊，靠远了漏风"（= 我也想抱着你）
- 她攥着香水瓶子不撒手（= 你送的东西我很珍惜）
- 她咬着牙说不冷，却把被子分他一半（= 比起自己冷更怕你冷）

## 四、多女主并行管理

### 同一场景中的空间分配
每个女主有自己的物理位置，男主在不同位置之间流动：
- 秦暮雪：房间/树荫（睡觉/逗猫）
- 薛凝：院子中央（练剑）
- 白初晴：灶台/后山（做饭/喂猫）
- 许落：在不同领地间穿梭

不能让所有女主站在一起说话。每人有自己的领地。

### 冷面角色的情感表达
适用对象：沈如霜、薛凝类角色

1. 极简对话+重动作：说"嗯""随便"，但做的是关键动作（让他枕腿上、深夜哭）
2. 反常即信号：平时绝不让人碰，某天让他握了手→读者马上知道有事
3. 读者比角色先察觉：男主不知道她哭了，读者知道→制造心疼效果
4. 关键词重复："我没有未来"→成为角色标签

### 各女主对话风格区分
确保读者仅通过对话就能分辨是谁在说话：
- 云浅：干脆利落，不废话，"阿越。"
- 苏晚晴：温柔但日渐坚定，"不必记得"/"我记得。"
- 沈如霜：极简，拒绝式，"不为什么。"
- 姬瑶光：傲娇嘴硬，"本宫只是路过。"
- 顾清寒：狡黠带笑，话中有话

## 五、冲突模式

### 微型摩擦（每章必有）
不是大矛盾，是小摩擦中露性格。参考小说每章示例：
- 要不要喝鱼汤→性格试探
- 她咬他手指→他冷脸→关系边界
- 他要走→她亮出修为→关系转折

### 社交博弈
不止打架。姜璎珞线全程无战斗但张力很强：
1. 她用暧昧当武器
2. 他先装傻后点破
3. 她被拆穿后威胁
4. 他不买账
连环博弈推进三章。

## 六、章末钩子五型

1. 关系即将变化："她忽然哭了"、"她轻轻搂住了他"
2. 悬念信息："他的脚边倒着几头刚刚毙命的妖兽"
3. 冲突预告："我说过让你走了吗？"
4. 幽默反转：许落以为安全了→姜璎珞出现在暮雪山
5. 事件启动："沧澜剑冢就要开放了"

钩子不需要是生死危机。关系的变化暗示比外部事件更有吸引力。

## 七、文笔禁令

以下在写正文时严格禁止：
1. 禁止使用"——"连接句子（场景分隔用空行）
2. 禁止大段心理独白（超过2句的"XX心里想"）
3. 禁止超过3句的环境描写
4. 禁止一段超过5句话
5. 禁止对话中的空洞寒暄（"你来了""嗯"）
6. 禁止物化女性表达（"护/放/守"等支配性动词描述男主对女主的感情）
7. 禁止外貌描写超过3句（出场时给2-3句，之后不再重复）

## 八、对话润色（写完自检）

写完章节后逐条检查：

### 书面化转口语化
书面："你怎么能这样对我"
口语："你什么意思" / "你——"（此处可用省略号"你……"）

书面："我绝不会让你失望的"
口语："放心"

### 空洞寒暄检测
标记并建议删除：
> "你来了。"
> "嗯。"
> → 删除，或加入信息量：
> "你怎么找到这的？"

### 对话动作穿插
连续3句以上纯对话无动作→插入小动作：
> "我不去。"
> "为什么？"
> "不为什么。"
> → 补充：
> "我不去。"她别过脸。
> "为什么？"
> 她沉默了一息。"不为什么。"

### 角色声音混淆检测
检查：不同角色的对话是否可以互换？如果能→声音区分不够。
- 每个角色有口头禅/习惯句式
- 冷面角色：句子最短，不用语气词
- 傲娇角色：开头否定，后面接让步
- 温柔角色：句子偏长，经常用"呢/呀"
```

## 第二步：写出正文

### 字数
每章4000-5000字。用短段落（1-3句为主）。

### 对话驱动
75%内容为对话或对话+动作。环境描写每处≤3句。

1. 开篇（5%）：承接上章，建立本章场景
