# 截图翻译工具 设计规格

**日期：** 2026-05-10
**状态：** 草稿

## 概述

一款 Windows 桌面应用，通过 F1 快捷键截取屏幕区域，对英文文字进行 OCR 识别，并将中文翻译结果以浮窗形式展示。

## 核心功能

1. **全局快捷键截图（F1）** — 按下 F1 进入区域选择模式
2. **OCR 文字识别** — 本地 Tesseract.js 引擎，英文模型，完全免费
3. **在线翻译** — 支持 DeepL Free API 或百度翻译 API（用户自行申请 Key 并选择使用哪个）
4. **译文浮窗展示** — 在截图区域旁边弹出置顶浮窗，展示原文+中文译文，支持复制和关闭

## 技术选型

- **框架：** Electron（纯 HTML/CSS/JS，不引入 React）
- **OCR：** tesseract.js（英文模型，本地运行）
- **翻译 API：** DeepL Free API（首选）、百度翻译 API（备选）
- **配置存储：** electron-store（本地 JSON 文件，API Key 加密存储）
- **打包：** electron-builder → Windows .exe / 便携版

## 架构

```
Electron 主进程
├── 注册全局快捷键 F1
├── 屏幕截图（desktopCapturer）
├── 区域选择遮罩层（透明遮罩 + 选区框）
├── 系统托盘（最小化到托盘，右键菜单：设置 / 退出）
└── IPC：将截取的图片传给渲染进程

Electron 渲染进程
├── 主窗口：设置界面（选择翻译服务、输入 API Key、测试按钮）
├── 译文浮窗：翻译结果展示
├── OCR 模块：tesseract.js worker，英文识别
└── 翻译模块：HTTP 调用 DeepL / 百度翻译 API
```

## 用户操作流程

1. 启动应用 → 打开主窗口
2. 首次使用：选择翻译服务，填入 API Key → 保存
3. 最小化到托盘 → 应用在后台监听 F1
4. 按 F1 → 屏幕冻结 → 用户拖拽选择截图区域
5. 松开鼠标 → 执行 OCR 识别（显示加载状态）
6. 调用翻译 API
7. 译文浮窗在截图区域旁边弹出
8. 用户：查看译文 / 复制 / 关闭浮窗

## 主窗口设计

- 尺寸：约 380×350，固定大小
- 翻译服务选择器：DeepL / 百度翻译（单选按钮）
- API Key 输入框：DeepL 模式显示单个 Key 输入框；百度模式显示 AppID + SecretKey 两个输入框（动态切换）
- [测试连接] 按钮：发送一个短文本试调翻译 API 验证 Key 是否有效
- [保存] 按钮：持久化配置
- 快捷键提示："F1 — 区域截图"
- 选项：开机自启复选框、最小化到托盘复选框
- [最小化到托盘] 按钮

## 译文浮窗

- 始终置顶，无标题栏，可拖动
- 每个识别到的文字块对应一组显示：
  - 原文（小号灰色字体）
  - 译文（大号加粗深色字体）
  - 文字块之间用分隔线隔开
- [复制] 按钮：将全部译文复制到剪贴板
- [关闭] 按钮：关闭浮窗
- 位置：在截图区域附近弹出

## 不包含的功能

- 图片编辑（裁剪、旋转、画笔、文字叠加）
- 截图贴到桌面
- 离线翻译
- 英文→中文以外的语言对
- 翻译历史记录
- 多显示器适配（v1 限定 — 仅支持主显示器）

## 依赖项

- `electron`、`electron-builder`
- `tesseract.js`（含英文训练数据包）
- `electron-store`
- DeepL REST API（`https://api-free.deepl.com/v2/translate`）
- 百度翻译 REST API（`https://fanyi-api.baidu.com/api/trans/vip/translate`）
